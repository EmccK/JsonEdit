# JSON 编辑器

可视化 JSON 编辑器，支持树形编辑、拖拽排序、实时预览。

## 功能

- 🌲 树形视图编辑 JSON 结构
- 🔀 拖拽排序节点
- ✏️ 双击编辑键名和值
- 📋 复制/删除/添加节点
- 📥 导入 JSON（粘贴或上传文件）
- 📤 导出 JSON（下载或复制）
- 🔍 搜索过滤
- ↩️ 撤销/重做 (Ctrl+Z / Ctrl+Shift+Z)
- 🎨 语法高亮预览

## 启动

```bash
npm install
npm run dev
```

## 技术栈

React + TypeScript + TailwindCSS + Zustand + dnd-kit
